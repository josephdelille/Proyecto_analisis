# Proyecto_analisis
Proyecto que realice para poner aprueba las habilidades que he adquirido como analista de datos

Usando SQL y realizando un análisis completo para determinar si los operadores de un centro de atención al cliente son eficientes o no, en este analisis yo mismo determine los parámetros con los que se evaluarían a los operadores y realice pruebas estadísticas para corroborar mis conclusiones
